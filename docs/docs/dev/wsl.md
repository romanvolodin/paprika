# Запуск Паприки в WSL для разработки

НЕ ПРОВЕРЕНО

## Подготовка

Установка WSL
```
wsl --install
```

По умолчанию скачивается и ставится Ubuntu. В дальнейшем запустить подсистему Линукс можно командой `wsl`. Будет запущен дистрибутив по умолчанию. Задать нужный дистрибутив можно командой `wsl -d {linux_name}`.

## Запуск

Передите в корень репозитория Паприки и запустить в текущей папке WSL:

```
wsl
```

Установите необходимые пакеты - git, make, docker, nodejs - [инструкция](../admin/prod-install.md).

Создайте переменные окружения:

```
make dev.init
```

Запустить Паприку:

```
make dev
```
